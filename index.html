Who We Are

从三维基础构建走向感知理解与决策行动

EngineeringAI Lab 相信，真正智能的系统必须能够理解、建模并与三维物理世界进行交互。 团队围绕高质量三维场景底座，开展 3D 基础表达与操纵、Agentic 空间理解与合成、 交互动态与运动、LLM 决策智能与推理等方向的研究。

我们将前沿研究与机器人仿真、具身智能、游戏影视等真实场景需求紧密结合， 推动 AI 从基础构建走向感知理解，并进一步迈向面向复杂任务的自主决策与行动。

Mission

我们的任务

01

3D Foundation

构建通用三维基础模型，覆盖从二维到三维的跨模态生成与理解， 支持高质量三维资产获取与结构化表达。

  • 图生 3D：从单张或多张图像重建高质量三维资产
  • 3D 分部件：实现部件级理解、编辑与重组
02

Agentic 感知与理解

研究 AI Agent 如何自主理解意图、调用工具并生成复杂三维场景， 同时构建面向空间认知的评测与仿真数据体系。

  • 基于 Agent 的三维场景自动构建
  • 基于引擎的场景理解工具与 Benchmark
03

交互动态与运动

面向数字人与多角色场景，建模真实可信的动态交互行为， 服务影视动画、游戏 NPC 与具身智能环境。

  • 2D/3D Talking Head：自然同步的表情与口型驱动
  • 多人运动交互：生成群体运动序列
04

LLM 决策智能与推理

探索 LLM 与强化学习、三维理解、任务规划的结合， 让智能系统面向复杂任务进行推理、决策与行动。

  • LLM Reasoning 与场景任务规划
  • 强化学习驱动的三维理解与交互

Work

我们做了什么

角色与人物生成

Xtalker

基于参数化建模的 2D Talking Head 系统，实现自然、同步的面部驱动生成。

角色与人物生成

3DXtalker

Xtalker 的三维延伸，构建具备空间一致性的 3D Talking Head 能力。

3D 基础模型

PyramidVoxel

面向三维内容生成的基础生成模型，采用金字塔体素结构提升质量与效率。

3D 基础模型

TOPS

基于几何结构的三维物体部件分解方法，实现结构感知的语义级拆解。

场景构建与理解

StoryBlender

基于 Agent 的三维场景自动构建系统，支持从叙事描述到完整场景的智能化生成。
